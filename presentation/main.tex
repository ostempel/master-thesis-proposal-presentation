\begin{frame}{Introduction}

\begin{itemize}
  \item Kubernetes and cloud-native technologies have become industry standard
  \item Key challenges: perceived complexity, lack of training \cite{cncf2024}
  \item Root cause analysis (RCA) as key bottleneck
\end{itemize}

$\Rightarrow$\ LLM-based assistance systems promise cognitive reduction and timely detection
